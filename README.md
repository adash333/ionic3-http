## Requirement

- Node.js
- Ionic CLI

## Installation

    $ git clone git@github.com:adash333/ionic3-http.git
    $ cd ionic3-http
    $ npm install @ionic/app-scripts@latest --save-dev
    $ npm install
    $ ionic serve

androidアプリの作成

    $ ionic platform add android
    $ ionic build android
